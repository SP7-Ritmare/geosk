GET-IT Central Data Node (CDN) of LTER-Europe network

enables you to provide time series data from field-based environmental sensors. You can describe sensor and upload data that are then linked to the eLTER portal as INSPIRE (2007/2/CE) web services.

To start uploading your data please authenticate yourself with DEIMS credential {% if user.is_authenticated %}and follows this steps:

  1. click on Sensor data -> Register a new Sensor menu
  2. fill out the template with the information of new sensor
  3. upload the observations in the Upload observations section
{% endif %}
